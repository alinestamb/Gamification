To set up json servers locally:
1. open another terminal
2. cd to file with json-servers (e.g., C:\Download\gamification\Gamification\gamification\src\json-servers)
2. npm i json-server
3. json-server --watch chapters.json --port 3001
4. open another terminal
5. cd to file with json servers
6. json-server --watch user.json --port 3002
